# Paper

IEEE-format conference paper for the Topics in Deep Learning project.

**Title:** *Uncertainty-Scaled Ensemble Transformer PPO for Risk-Controlled
Multi-Asset Trading under Market Impact and Inferred Hidden Liquidity*

## Layout
- `main.tex` — entry point, IEEEtran two-column conference class.
- `references.bib` — BibTeX entries (seeded; expand in D18 Section II).
- `figures/` — symlink to project-root `figures/`. Plots are generated by
  `scripts/make_figures.py`; the architecture diagram is built from
  `figures/architecture.tex`.

## Build
```
cd paper
pdflatex architecture.tex   # one-time, produces architecture.pdf in figures/
pdflatex main.tex
bibtex   main
pdflatex main.tex
pdflatex main.tex
```

`pdflatex` is **not installed locally** (verified 2026-04-22). Install
MacTeX (`brew install --cask mactex-no-gui`) or BasicTeX, or compile via
Overleaf by uploading `paper/` plus the symlink target `figures/`.
